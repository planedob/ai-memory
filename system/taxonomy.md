# Taxonomy

统一标签有助于多个 agent 写出兼容的数据。

## type

- `fact`: 稳定事实
- `preference`: 用户偏好或约束
- `behavior`: agent 行为
- `decision`: 已做出的选择
- `task`: 待办或行动项
- `risk`: 风险或阻塞
- `note`: 普通备注

## confidence

- `high`: 已验证或直接来源
- `medium`: 有较强依据但未完全验证
- `low`: 推测、猜测、待确认
