# Workflow Reference

## 1. ADX / Competitor collection schema

For each collected creative, capture:
- Platform and placement
- Product name
- Genre / subgenre
- Country / language
- Creative URL or internal asset id
- Format: video / image / playable / carousel
- Duration
- First-frame description
- First-3-second hook
- Core promise
- Emotional trigger
- Apparent audience
- Gameplay truth level: real / hybrid / exaggerated / fake
- Monetization or payoff shown
- CTA style
- Visual motifs
- Copy / subtitle transcript
- Repeated market pattern tag
- Notes on why it may be working

## 2. Questions for operations/product

Ask for:
- Real gameplay loop in one sentence
- Most satisfying progression moment
- Strongest novelty or differentiator
- Monetization moment worth previewing
- Features we can promise truthfully
- Features we must not fake
- Current retention or payer insights
- Upcoming version changes that affect creatives

## 3. Questions for market/growth

Ask for:
- Target countries and platforms
- Core audience clusters
- CPI / CPA / ROI / ROAS targets
- Payback window
- Existing high-performing angles
- Fatigued angles to avoid
- Priority placements and budgets
- Whether the goal is volume, stable ROI, or scaling winners

## 4. Questions for media buying / creative ops

Ask for:
- Current winning creatives and why they win
- Main failing patterns and suspected causes
- Hook-through rate issues vs conversion issues
- Learning-stage blockers
- Spend caps and batch size
- Editing capability and asset constraints
- Need for localization or multi-language output

## 6. Production pack template

For each concept provide:
- Concept id
- Hook family
- Audience hypothesis
- Reference pattern(s)
- Differentiation note
- 3-second opener
- 15s script
- 30s script
- Shot list
- Subtitle lines
- VO lines
- CTA options
- Needed assets
- Editing notes
- Platform notes

## 7. Post-launch diagnosis checklist

Check in this order:
1. Is the audience wrong or the hook weak?
2. Is CTR poor because the opener misses?
3. Is click strong but install/lead weak because the promise mismatches the page or store?
4. Is install strong but payer/ROI weak because the wrong users are being attracted?
5. Is fatigue caused by market saturation or over-repetition of the same angle?

## 8. Default collaboration rule

When the user asks for game or novel ad materials, assume a three-way collaboration:
- Market owns audience and KPI context.
- Operations owns product truth and real hooks.
- Creative/media buying owns market intelligence and execution.

The skill should actively surface missing inputs from any of the three sides before finalizing a strategy.
